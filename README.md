## CORS 에러 해결

- 클라이언트에서 API 요청 시 발생하는 CORS 에러를 해결하기 위해 만든 작은 서버입니다.
- Express로 API 요청을 받아오고 CORS 라이브러리를 사용하여 CORS 에러가 발생하지 않도록 합니다.
